# TDetect v1.0

## Requirements for Installation (Video):
- Python packages:
  opencv-python, torch, torchvision, torchaudio  
  Install with:  
  pip install opencv-python torch torchvision torchaudio

- Seaborn:  
  Install with:  
  pip install seaborn

- Ultralytics:  
  Install with:  
  pip install ultralytics

  ## Requirements for Installation (Real Time):
- Python packages:
  opencv-python, torch, torchvision, torchaudio  
  Install with:  
  pip install opencv-python torch torchvision torchaudio

- Seaborn:  
  Install with:  
  pip install seaborn

- Ultralytics:

  pip install mss
  
- PyQt5:
